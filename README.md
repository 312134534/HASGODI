# HASGODI

Unity 遊戲專案 - HASGODI

## 遊戲介紹

HASGODI 是一款由 Unity 開發的 2D 雙人格鬥遊戲，可以雙人對戰或挑戰電腦。

## 遊玩方式

### Player1
- A / D：左右移動  
- J：近攻擊  
- U：遠攻擊擊
- K：跳躍  
- L：閃避
### Player2
- <- / ->：左右移動  
- 1：近攻擊  
- 4：遠攻擊擊
- 2：跳躍  
- 3：閃避

### UI
- Esc：暫停選單
